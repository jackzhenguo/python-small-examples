64 更长列表
-----------

.. code:: python

   def max_length(*lst):
       return max(*lst, key=lambda v: len(v))


   r = max_length([1, 2, 3], [4, 5, 6, 7], [8])
   print(f'更长的列表是{r}')  # [4, 5, 6, 7]

   r = max_length([1, 2, 3], [4, 5, 6, 7], [8, 9])
   print(f'更长的列表是{r}')  # [4, 5, 6, 7]
