#### 76 Topn 字典

返回字典d前n个最大值对应的键

```python
from heapq import nlargest
def topn_dict(d, n):
    return nlargest(n, d, key=lambda k: d[k])
```

测试：

```python
topn_dict({'a': 10, 'b': 8, 'c': 9, 'd': 10}, 3)  
# ['a', 'd', 'c']
```
