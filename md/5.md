#### 5 至少一个为真检查　

至少有一个元素为真返回`True`，否则`False`
```python
In [7]: any([0,0,0,[]])                                                         
Out[7]: False
```

```python
In [8]: any([0,0,1])                                                            
Out[8]: True
```
