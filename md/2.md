#### 2  进制转化

十进制转换为二进制：
```python
In [2]: bin(10)                                                                 
Out[2]: '0b1010'
```

十进制转换为八进制：
```python
In [3]: oct(9)                                                                  
Out[3]: '0o11'
```

十进制转换为十六进制：
```python
In [4]: hex(15)                                                                 
Out[4]: '0xf'
```
