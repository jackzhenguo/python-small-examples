#### 2  进制转化

十进制转换为二进制：
```python
In [1]: bin(10)                                                                 
Out[1]: '0b1010'
```

十进制转换为八进制：
```python
In [2]: oct(9)                                                                  
Out[2]: '0o11'
```

十进制转换为十六进制：
```python
In [3]: hex(15)                                                                 
Out[3]: '0xf'
```
